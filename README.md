# 🗂️ Python CLI File Manager

A command-line based **File Management System** built in Python.  
This project allows you to browse, copy, move, delete, search, backup files,  
download content from the internet, and check your system info — all from one CLI menu.

---

## 🚀 Features
- 📂 **Browse Files**: View files inside the project folder  
- 📋 **Copy / Move Files**: Copy or move files with duration tracking  
- ❌ **Delete Files/Folders**: Safely delete items  
- 🌐 **Download Files**: Download any file from a given URL  
- 🗄️ **Backup System**: Create backups of files/folders  
- 🔍 **Search Files**: Search for files inside any directory  
- 💻 **System Info**: Shows CPU, RAM, and Disk usage  
- ⏱️ **Performance Tracking**: See how long each operation takes  
- ⚡ **Error Handling**: Prevents program from crashing  

---

## 🛠️ Tech Stack
- **Python 3.x**
- Libraries used:
  - `os` → File & directory operations  
  - `shutil` → File copy/move/delete  
  - `time` → Duration tracking  
  - `psutil` → System information  
  - `requests` → Download files

## ⚡ How to Run
1. Clone this repository:
   ```bash
   git clone https://github.com/your-username/python-cli-file-manager.git
   ```
2. Navigate into the folder:
   ```bash
   cd python-cli-file-manager
   ```
3. Run the project:
   ```bash
   python main.py
   ```
